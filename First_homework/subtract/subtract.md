# 超长正整数的减法

**【问题描述】**
编写程序实现两个超长正整数（每个最长80位数字）的减法运算。

**【输入形式】**
从键盘读入两个整数，要考虑输入高位可能为0的情况（如00083）。

1. 第一行是超长正整数A；
2. 第二行是超长正整数B；

**【输出形式】**
输出只有一行，是长整数A减去长整数B的运算结果，从高到低依次输出各位数字。要求：若结果为0，则只输出一个0；否则输出的结果的最高位不能为0，并且各位数字紧密输出。

**【输入样例】**

```text
234098
134098703578230056

```

**【输出样例】**

```text
-134098703577995958

```

**【样例说明】**
进行两个正整数减法运算，234098 －134098703578230056 = －134098703577995958。
